# Memory-card-game-with-C-
A memory card game made using C++
Bu proje, **C++ dili** ve **Qt Framework** kullanılarak geliştirilmiş bir hafıza kartı oyunudur. Oyuncuların aynı resimli kartları eşleştirmesi amaçlanır.


## Özellikler

- 10 adet kart (5 çift) ile oynanır.
- Kartlar başlangıçta kapalıdır ve tıklanınca açılır.
- Açılan iki kart eşleşiyorsa açık kalır, değilse tekrar kapanır.
- Oyuncunun başarısız tahmin sayısı takip edilir.
- Tüm kartlar eşleştiğinde oyun biter.



## Kullanılan Teknolojiler

- **Programlama Dili:** C++
- **IDE Önerisi:** Qt Creator



## Kurulum ve Çalıştırma

1. Qt Creator veya uygun bir C++ IDE açın.
2. Bu projeyi Qt Creator ile açın.
3. Projeyi derleyin ve çalıştırın (`Ctrl+R` veya `Run`).
